# Hexo blog theme
this is hexo blog theme. [Live demo](http://hikonaz.com).
WIP as public theme.

## requirement

```
    "hexo-renderer-jade": "0.2.0"
```

Using `hexo-renderer-jade` `v0.2`, cause `v0.3` is not working with livereload.
