# Hexo blog theme
this is hexo blog theme.

## requirement

```
    "hexo-renderer-jade": "0.2.0"
    "hexo-generator-sitemap": "^1.1.2"
```

I'm using `hexo-renderer-jade` `v0.2`, cause `v0.3` is not working with livereload.
